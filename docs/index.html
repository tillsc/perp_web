Regatta-Management,
das einfach funktioniert.

Von der Meldung bis zur Siegerehrung – alles in einem Browser-Tab. Keine teure Hardware, keine Installation, kein Papierchaos.

Auf GitHub ansehen

Alles, was eine Regatta braucht

perp.de begleitet eine Ruder-Regatta von Anfang bis Ende – und alle Beteiligten arbeiten gleichzeitig von ihrem eigenen Gerät aus.

Meldungen & Einteilung

Nennungen importieren oder manuell erfassen, Startbahnen zuweisen, Startlisten drucken. Nachimporte aus dem DRV-Portal sind jederzeit möglich, ohne bestehende Daten zu überschreiben.

Zeitplan & Rennprogramm

Geplante Startzeiten verwalten und das Rennprogramm grafisch als Zeitstrahl darstellen. Zeitblöcke lassen sich flexibel verschieben – der Zeitplan bleibt jederzeit aktuell.

Zeitnahme

Manuelle Eingabe auf Handy oder Tablet – auf Wunsch mit günstigem Bluetooth-Taster als Auslöser. Oder vollständig automatisch per Kamerasystem, das sich am Ziel und an beliebigen Zwischenpunkten auf der Strecke einsetzen lässt.

Ergebnisse in Echtzeit

Sobald ein Ergebnis eingetragen ist, sehen es alle: Kampfrichter, Vereinsvertreter und das TV-Display im Zielbereich – ohne Extra-Schritte.

Schlitzkamera-System

Das Zusatzmodul perp_finish_cam erzeugt präzise Schlitzbilder für die Zielauswertung. Es benötigt eine USB-Kamera mit Teleobjektiv – kein Profi-Gerät, aber ein Tele ist nötig – und läuft auf jedem handelsüblichen Laptop. Optional erkennt eine KI automatisch die Bugposition der Boote.

TV-Anzeigesystem

Mehrere Anzeigemodi (aktueller Start, letztes Rennen, Sieger, Informationsfolien) – steuerbar vom Ansager-Pult. Schriften und Farben frei einstellbar.

Drucken aus dem Browser

Startlisten, Ergebnisblätter und Rennprogramme lassen sich direkt aus dem Browser drucken – mit sauberem Layout, automatischen Kopf- und Fußzeilen auf jeder Seite und korrekten Seitenumbrüchen. Kein separates Druckprogramm nötig.

Vereinsportal

Jeder Verein bekommt einen privaten Link und kann damit seine eigenen Meldungen, Starts und Ergebnisse einsehen – ohne Anmeldung, auf jedem Gerät.

Auf einen Blick

Was die Software mitbringt.

  • Läuft vollständig im Browser – keine Installation nötig
  • Mobil, Tablet und Desktop gleichzeitig nutzbar
  • Direkter Import aus dem DRV-Meldeportal, beliebig wiederholbar
  • Grafischer Zeitplan mit flexiblen Zeitblöcken
  • Manuelle Zeitnahme auf Handy/Tablet, optional mit Bluetooth-Taster
  • Schlitzkamera-System an Ziel und Zwischenpunkten (perp_finish_cam)
  • Mehrere Messpunkte: Start, Zwischenzeiten, Ziel
  • Live-Ranglisten mit Zeitabständen
  • TV-Anzeigesystem für Videoanlage oder Beamer
  • Gewichtsprüfung für Leichtgewichts-Rennen
  • Rollenbasierter Zugang: Zeitnehmer, Schreiber, Kampfrichter, Ansager …
  • Drucklayouts direkt aus dem Browser – Startlisten, Ergebnisse, Programm
  • Ergebnis-Export im DRV-Format (XML)
  • Open Source – selbst betreibbar oder per Docker deployen

Was es von klassischen Lösungen unterscheidet

Die meisten Regatten werden heute noch mit einer Mischung aus proprietärer Hardware, Tabellen-Kalkulation und Papier organisiert. perp.de löst das anders.

  • Keine teure Spezialhardware Statt Zeitnahme-Anlagen für mehrere tausend Euro reicht ein Handy oder Tablet im Browser – mit günstigem Bluetooth-Taster als Auslöser. Das Schlitzkamera-System braucht nur einen handelsüblichen Laptop und eine USB-Kamera mit Teleobjektiv.
  • Alles vernetzt, in Echtzeit Zeitnehmer, Schreiber, Kampfrichter und Ansager arbeiten gleichzeitig im selben System. Kein Zettelübergeben, kein Warten auf den Ausdruck vom Nebentisch.
  • Echte Verbandsdaten-Integration Der Import aus dem DRV-Meldeprogramm lässt sich beliebig oft wiederholen – Nachkorrekturen des Verbandes landen automatisch im System, ohne dass bereits eingetragene Daten verloren gehen.
  • Läuft auf jedem Gerät Wer einen Browser hat, kann mitmachen. Kein Installations-Stress am Wettkampfmorgen, keine Windows-only-Software.
  • Skaliert mit der Veranstaltung Das System wurde für große Regatten entwickelt und bewährt sich dort – funktioniert aber genauso zuverlässig bei kleineren Vereinsregatten.
